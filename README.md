# Glassmorpher

Generador de estilos glassmorphism y neumorphism para crear tarjetas y componentes con look moderno sin tunear CSS a ciegas.

- Features
  - Controles de blur, opacidad, borde y color base
  - Modos glassmorphism y neumorphism con presets ajustables
  - Export de CSS listo para pegar o convertir en componente reutilizable

- Stack
  - React
  - TypeScript
  - Tailwind CSS
